⚡ QUICKSTART (10 MINUTOS):

PASO 1: Token de GitHub
  1. Entra a https://github.com/settings/tokens
  2. "Generate new token (classic)"
  3. Selecciona "repo"
  4. Copia el token

PASO 2: Subir a GitHub
  1. Abre VS Code
  2. Ctrl+Shift+G (Source Control)
  3. "Initialize Repository"
  4. Stage All
  5. Commit: "Initial: Editor Pro 2 v2.0"
  6. Publish Branch
  7. Pega el token

PASO 3: Conectar Railway
  1. Entra a https://railway.app
  2. Login con GitHub
  3. "New Project" → "Deploy from GitHub"
  4. Selecciona "editorpro2"
  5. Deploy

PASO 4: Configurar Variables
  1. En Railway: Tu proyecto → Variables
  2. Agrega:
     PORT = 3000
     NODE_ENV = production
     LM_STUDIO_URL = http://tu-servidor:1234/v1/chat/completions
  3. Save

LISTO! 🎉 Tu app está en https://editor-pro2-production.up.railway.app
